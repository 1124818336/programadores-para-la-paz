Al enviar una petición POST a http://localhost:3000/registro con el JSON de prueba, el servidor procesa la información entrante a través de req.body. Como respuesta, devuelve un objeto JSON confirmando el éxito de la operación. La respuesta exacta que devuelve el servidor es:

{
  "estado": "Datos recibidos",
  "nombre": "Maria",
  "mensaje": "Hola comunidad"
}
